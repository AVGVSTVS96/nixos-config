## Darwin (macOS)
Configuration modules specific to macOS systems using nix-darwin.

## Layout
```
.
├── dock/              # MacOS dock configuration
├── files.nix          # Symlinks for non-nix config files
├── home-manager.nix   # Home-manager configuration and dock configuration for macOS
├── homebrew.nix       # Homebrew package management, casks and App Store apps
├── packages.nix       # List of additional packages to install for macOS
└── README.md          # This file
```
